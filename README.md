# ✈️ Aircraft Trajectory Anomaly Detection using Machine Learning

## Overview

This project focuses on detecting anomalous aircraft trajectories using machine learning techniques to improve aviation safety and enable early conflict warning.

Developed as part of my MSc in Advanced AI & Data Science at the University of Liverpool.

---

## Objectives

- Collect aircraft trajectory data from OpenSky Network
- Preprocess continuous flight data
- Perform feature engineering
- Detect anomalous flight behaviour
- Compare multiple machine learning models

---

## Models

- Isolation Forest
- LSTM Autoencoder
- Random Forest (Baseline)
